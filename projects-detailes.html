Project Overview

This full-stack financial application leverages AI and machine learning to process data from Google Sheets, perform predictive analysis, and automatically generate comprehensive PDF reports. It's built for businesses needing quick insights into financial trends and forecasting.

Key Features

  • Data ingestion from Google Sheets API
  • AI/ML model training for financial predictions
  • Automated PDF report generation with charts
  • Secure user authentication and dashboard
  • Real-time data visualization using React

Challenges & Solutions

Challenge: Integrating Google Sheets API

Handling real-time data pulls and authentication issues with varying sheet permissions.

Solution: OAuth & Error Handling

Implemented robust OAuth flows and fallback mechanisms for data sync failures.

Challenge: AI Model Training Efficiency

Training ML models on large datasets without excessive compute time.

Solution: Optimized Python Scripts

Used TensorFlow Lite for lighter models and scheduled batch processing.

Challenge: PDF Report Customization

Generating dynamic, branded PDFs with embedded charts.

Solution: Puppeteer Integration

Leveraged Puppeteer for HTML-to-PDF conversion with custom styling.
